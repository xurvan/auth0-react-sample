# Auth0 React sample

## Prerequisites
- Node.js
- moonrepo proto
- pnpm package manager

## Setup and Installation

### 1. Install Dependencies
```bash
proto install
pnpm install
```

### 2. Auth0 Configuration
In the `main.jsx` file you can chance these:
- `domain`
- `clientId`
- `audience`


### 4. Running the Project
```bash
pnpm run dev
```
